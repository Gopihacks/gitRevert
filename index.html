git branch =>to list all the branches in your repository. The current branch will be highlighted with an asterisk (*). git branch -a=> to list all branches, including remote branches. git branch => to create a new branch with the specified name. git checkout => to switch to the specified branch. (or) git switch => to switch to the specified branch. git checkout -b => to create a new branch and switch to it immediately. (or) git switch -c => to create a new branch and switch to it immediately first switch to main branch then git merge => to merge the specified branch into the current branch. git branch -d => to delete the specified branch.In local repository git push origin --delete => to delete the specified branch in remote repository. git pull origin => to fetch and merge changes from the specified branch in the remote repository to your local branch.
